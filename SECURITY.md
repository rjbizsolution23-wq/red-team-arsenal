# Security Policy

## Supported Versions

Only the latest `main` branch version is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Yes             |
| < 1.0   | ❌ No              |

## Reporting a Vulnerability

**DO NOT OPEN PUBLIC ISSUES FOR SECURITY VULNERABILITIES.**

If you discover a potential security flaw in the Red Team Arsenal, please report it privately via email to:

**rjbizsolution23@gmail.com**

Please include:
- A clear description of the vulnerability.
- Steps to reproduce (POC).
- Any potential impact or suggested fixes.

We aim to respond within 48 hours and coordinate a fix before public disclosure.
